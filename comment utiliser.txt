Comment utiliser SaCorpUpdater :
- Uploader le dossier serveur sur votre FTP
- Placer les fichiers SaCorpUpdater.exe, SaCorpUpdater.exe.manifest, unrar.dll et Config\Launcher.ini à la racine de votre programme
- Ouvrir le fichier Config\Launcher.ini et modifiez-le.

-------------

Comment ajouter une version :
Allez dans le dossier Serveur de votre FTP.
Ouvrez le fichier update.ini :
[FILES]
versions=Le nombre de versions de votre programme
version1=version1.rar
version2=version2.rar
etc... en fonction du nbre de versions

et dans le même dossier, créer une archive .rar avec les fichiers MAJ dedans.
